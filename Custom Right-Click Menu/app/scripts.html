Sharing and browsing scripts



Here you can both share scripts yourself, and use scripts made by other people.
I will look through every script that is sent to make sure nothing malicious is in there, however i can not guarantee that nothing bad will happens.
These are some things you can do if something malicious does happen though:

One thing it can do is create an infinite loop, if this happens simply shut down your PC.
It will be pretty hard to shut it down the normal way with all the windows popping up, so just hold down the power button untill it shuts down.
When you have important work still waiting to be saved, just try and save that first. Another way to close chrome is to go to task manager and close it via services. This is possible but way harder than doing it the power-off-button way. Due to how this extension works, nothing can happen unless you launch the script again. Simply remove the script, and report it to me if you want to help other people.

Scripts should not be able to send any data to other sites. This means that any data collected will not be available for them as long as they don't get acces to your computer. Scripts CAN collect data from websites, even passwords. It's very easy to do this when a script is running in your browser. But they can not send the data anywhere due to chrome's extension's limitations.

I can not think of anything malicious a script could do except from these, but that's what people who try to make malicious scripts do, create them.

Unfortunately, this method of hosting the scripts doesnt allow comments, numer of installs or ratings which would help prevent these problems. If you've got any suggestions for a place to host these, please contact me. I'll try to set up my own server if this extension gets a lot of users, i'm not going to create a server for ~30 people using it since there will barely be any scripts.
