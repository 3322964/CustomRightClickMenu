CRM API Docs
Welcome to the CRM API Docs page. Here you can read up on the CRM API's functions and properties to make you better at using it. You can navigate this page by going to any of the sections on the left where each section represents one part of the API (the GM section represents crmapi.GM etc). Additionally by using the default editor in CRM and pressing the "Show Docs" key (default Ctrl-O), you can see any docs for the current function in-editor or go to its documentation by clicking the link to it.

data types
Data types used in the CRM API apart from the default javascript ones

crmNode
The value of a standard node, all nodes inherit from this


Properties:
NumberidThe ID of the node
NumberindexThe index of the node in its parent's children
StringnameThe name of the node
StringtypeThe type of the node (link, script, menu or divider)
Array of crmNodeschildrenThe children of the object, only possible if type is menu and permision "CRM" is present
ObjectnodeInfoAny info about the node, it's author and where it's downloaded from
StringnodeInfo.installDateThe date on which the node was installed or created
BooleannodeInfo.isRootWhether the node is downloaded (false) or created locally (true)
Array of StringsnodeInfo.permissionsPermissions required by the node on install
String or ObjectnodeInfo.source'Local' if the node is non object if it IS remotely installed
StringnodeInfo.source.urlThe url that the node was installed from
Array of NumberspathThe path to the node from the tree's root
Array of BooleansonContentTypesThe content types on which the node is visible there's 6 slots, for each slot true indicates it's shown and false indicates it's hidden on that content type, the content types are 'page','link','selection','image','video' and 'audio' respectively
Array of StringspermissionsThe permissions required by this script
Array of ObjectstriggersThe triggers for which to run this node
Stringtriggers.urlThe URL of the site on which to run, according to the chrome match patterns found at https://developer.chrome.com/extensions/match_patterns
Booleantriggers.notIf true does NOT run on given site
CrmAPIInit~linkVallinkValThe value of the node if it were to switch to type link
CrmAPIInit~scriptValscriptValThe value of the node if it were to switch to type script
CrmAPIInit~stylesheetValstylesheetValThe value fo the node if it were to switch to type stylesheet
Array of ObjectsmenuValThe children of the node if it were to switch to type menu

linkVal
The properties of a node if it's of type link


Properties:
Array of ObjectsvalueThe links in this link
Stringvalue.urlThe URL to open
Booleanvalue.newTabTrue if the link is opened in a new tab
BooleanshowOnSpecifiedWhether the triggers are actually used, true if they are

Inherits from:
crmNode

scriptVal
The properties of a node if it's of type script


Properties:
ObjectvalueThe value of this script
Numbervalue.launchModeWhen to launch the script, 0 = run on clicking 1 = always run 2 = run on specified pages 3 = only show on specified pages
Stringvalue.scriptThe script for this node
Stringvalue.backgroundScriptThe backgroundscript for this node
Objectvalue.metaTagsThe metaTags for the script, keys are the metaTags, values are arrays where each item is one instance of the key-value pair being in the metatags
Array of Objectsvalue.librariesThe libraries that are used in this script
scriptvalue.libraries.nameThe name of the library
Array of Objectsvalue.backgroundLibrariesThe libraries that are used in the background page
scriptvalue.backgroundLibraries.nameThe name of the library

Inherits from:
crmNode

stylesheetVal
The properties of a node if it's of type stylesheet


Properties:
ObjectvalueThe value of this stylesheet
Numbervalue.launchModeWhen to launch the stylesheet, 0 = run on clicking 1 = always run 2 = run on specified pages 3 = only show on specified pages
Stringvalue.stylesheetThe script that is ran itself
Booleanvalue.toggleWhether the stylesheet is always on or toggleable by clicking (true = toggleable)
Booleanvalue.defaultOnWhether the stylesheet is on by default or off, only used if toggle is true

Inherits from:
crmNode

menuVal
The properties of a node if it's of type menu


Properties:
BooleanshowOnSpecifiedWhether the triggers are actually used, true if they are

Inherits from:
crmNode

dividerVal

Properties:
BooleanshowOnSpecifiedWhether the triggers are actually used, true if they are

Inherits from:
